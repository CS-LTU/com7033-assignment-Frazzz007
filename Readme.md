# Stroke Patient Management System

A secure Flask web application for managing stroke patient records.

## Features
- Doctor login authentication
- Add, edit, delete patient records
- MongoDB for patient data
- SQLite for user authentication
- CSRF Protection
- Password hashing
- Pagination & search

## How to Run
1. Create virtual environment
2. Install requirements
3. Run app.py

## Security
- CSRF protection
- Password hashing
- Login required routes
